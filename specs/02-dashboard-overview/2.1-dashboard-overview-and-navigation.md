# 2.1 | Dashboard overview and navigation

Provides unified command centre displaying IP portfolio status, critical deadlines, team activity and quick access to all platform functions through intuitive navigation.

## Segmentation / user types
 All user types submitting IP applications

- IP Creator

- IP Manager,

- IP Legal Advisor

- IP Consultant

## Outcomes

- View unified dashboard of IP portfolio and applications

- Navigate to core platform functions easily

- Track application progress and status

- Access quick start options for new applications

## User Stories

### MVP Scope

- When I log into the platform, I want to see all my applications in one unified view so I can quickly assess my portfolio status

- When starting new applications, I want quick access to creation tools so I can begin immediately.

- When managing multiple IP assets, I want to track progress of all applications so I can ensure nothing is overlooked.

- When deadlines approach, I want prominent alerts so I can take timely action.

- When navigating the platform, I want intuitive organisation so I can find features quickly.

- When managing my IP portfolio, I want to access enforcement guides so I can understand my rights and take action against infringement across Patents, Trademarks and Copyright.

- When reviewing my IP portfolio, I want to access commercialisation guides so I can generate revenue from my Patents, Trademarks and Copyright through licensing and strategic partnerships.


# Acceptance criteria draft

## Job Story 1: Unified portfolio view

When I log into the platform, I want to see all my applications in one unified view so I can quickly assess my portfolio status

### Acceptance criteria:

Dashboard displays all IP applications in a single interface

Portfolio status shows total asset count and breakdown by type (patents, trademarks, copyright)

Application progress is visible for each active application

Status indicators clearly show application states (in progress, submitted, under review, action required)

Portfolio statistics update when new applications are added

## Job Story 2: Quick access to registration tools

When I need to start a new application, I want quick navigation to registration tools so I can begin work without searching

### Acceptance criteria:

"Register Patent", "Register Trademark", "Register Copyright" buttons are prominently displayed

Quick actions bar provides immediate access to new application creation

Navigation to registration tools requires maximum 3 clicks

New application modal opens with clear asset type selection options

## Job Story 3: Application progress tracking

When managing multiple IP assets, I want to track progress of all applications so I can ensure nothing is overlooked

### Acceptance criteria:

Applications in progress section displays all active applications

Progress indicators show completion percentage for each application

Last updated dates are visible for all applications

Applications can be filtered by status and type

Empty states provide guidance when no applications exist

## Job Story 4: Deadline alerts

When deadlines approach, I want prominent alerts so I can take timely action

### Acceptance criteria:

Alert banner displays urgent notifications at top of dashboard

Office action deadlines show countdown in days

Renewal deadline notices appear with adequate lead time

Alert count badge indicates number of pending notifications

High priority alerts are visually distinct from standard notifications

## Job Story 5: Intuitive navigation

When navigating the platform, I want intuitive organisation so I can find features quickly

### Acceptance criteria:

Primary navigation provides clear access to core domains (Portfolio, Registration, Enforce, Commercialise)

Platform maintains consistent navigation structure across all pages

User location within platform is clearly indicated

Search functionality enables quick asset discovery

Help and support resources are easily accessible from any page

## Job Story 6: Enforcement guidance access

When managing my IP portfolio, I want to access comprehensive enforcement guides so I can understand my rights

### Acceptance criteria:

Expert guides section includes enforcement guidance for patents, trademarks, and copyright

Enforcement information explains rights and available actions for each IP type

Guidance is accessible directly from dashboard without complex navigation

Content is appropriate for non-native English speakers

## Job Story 7: Commercialisation guidance access

When reviewing my IP portfolio, I want to access commercialisation guides so I can generate revenue

### Acceptance criteria:

Expert guides section includes commercialisation information for all IP types

Licensing and strategic partnership guidance is clearly presented

Revenue generation options are explained for patents, trademarks, and copyright

Content supports international business practices
